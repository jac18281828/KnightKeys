KnightKeys - a fast program to compute number of allowed sequences for "knight moves" on a keypad

The default argument is ten, but optionally max path length may be specified on the command line:

$ java -cp dist/lib/KnightKeys-20120301.jar com.cairns.knightkeys.KnightKeys
1013398

$ java -cp dist/lib/KnightKeys-20120301.jar com.cairns.knightkeys.KnightKeys 10
1013398
